# EEG_Pipeline_Speech_Learning_Perception
A EEG preprocessing and analysis pipeline / libraries on sine - wave speech perception study.
Runtime Environment Requirements (Conda based): 

  MNE 1.11.0, 
  
  statsmodels 0.14.5, 
  
  scikit-learn 1.7.2.

  

Verified only on the aforementioned version.


*Note*: The demo data we prepared for code validation exceeds the repository's capacity... 

If you require the demo data, please visit the OSF repository or contact the authors.
